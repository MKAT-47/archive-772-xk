⚠ Known Limitations & Notes
Save File button uses outerHTML — some browsers may strip event listeners. Data is always embedded and will reload correctly.
Draft system tracks priority via _draftPriority field — not yet visible in the main character view header (planned for v17).
Relationship Web does not yet update in real time — requires opening the web view after editing characters.
Location view tabs 4–6 (Society, Infrastructure, Connected) require at least one location to be saved before data appears correctly.
The codex is entirely offline — no cloud sync. Use the 💾 Save File button regularly and keep copies in multiple places.
